# 岗位二：高级全栈工程师 Java
## 基础信息
- 工作地点：东营
- 薪资：15-20K
- 工作年限：5-10年
- 学历：学历不限
- 标签：TypeScript、Java、Spring/Spring Boot、JavaScript、全栈无侧重、MySQL/Redis/MongoDB、全栈项目经验
## 岗位职责
1. Java核心能力：精通Java语言及JDK核心API，深入理解JVM内存模型、垃圾回收机制及类加载机制，具备5年以上Java开发经验。
2. 微服务架构：熟练掌握微服务架构设计与落地，精通Spring Cloud/Alibaba生态组件（如Nacos、Sentinel、Gateway等），能独立完成微服务拆分、服务治理及高可用设计。
3. 硬件MQTT对接：具备丰富的硬件设备MQTT协议对接经验，能处理设备连接、数据传输、消息订阅/发布等全流程，解决低功耗、高并发场景下的通信稳定性问题。
4. Spring Boot调优：精通Spring Boot框架原理，能从源码层面分析并优化Bean加载、自动配置、性能瓶颈等，熟练运用监控工具（如Prometheus、Grafana）完成应用性能调优。
5. 数据库调优：深入掌握MySQL等关系型数据库，精通索引优化、SQL调优、事务隔离级别及锁机制，具备分库分表、读写分离等数据库架构设计与调优经验。
6. 中间件与工具：熟练使用Redis、RabbitMQ/Kafka等中间件，掌握Git、Maven、Jenkins等开发与部署工具，具备Docker、Kubernetes等容器化部署经验者优先。
7. 项目经验：有大型分布式系统或高并发业务项目开发及优化经验，能独立负责核心模块设计与开发，具备良好的问题排查和技术攻坚能力。
